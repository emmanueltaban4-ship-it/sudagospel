# Project Memory

## Core
Sudagospel.net — gospel music platform for South Sudan. Primary red (#DC2626), secondary green, dark tones.
Plus Jakarta Sans headings, DM Sans body. Mobile-first with bottom nav.
Lovable Cloud enabled. Auth via email/phone. Storage: music, covers, avatars, ads buckets.
Admin system uses user_roles table with has_role() security definer function.

## Memories
- [Branding](mem://design/branding) — Red/green/dark palette from South Sudan flag colors, gospel theme
- [Database schema](mem://features/database) — profiles, artists, songs, likes, comments, downloads, ads, user_roles tables with RLS
