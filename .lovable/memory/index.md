# Project Memory

## Core
Sudagospel.net — gospel music streaming for South Sudan. Primary gold (#D4A017), secondary purple (#7C3AED), dark background.
Figtree headings, DM Sans body. Spotify/Audiomack-inspired UI. Mobile-first with bottom nav.
Lovable Cloud enabled. Auth via email/Google. Storage: music, covers, avatars buckets.
Admin system uses user_roles table with has_role() security definer function.
