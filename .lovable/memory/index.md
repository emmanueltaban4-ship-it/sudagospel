# Project Memory

## Core
Sudagospel.net — gospel music platform for South Sudan. Primary purple, accent gold, dark tones.
Plus Jakarta Sans headings, DM Sans body. Mobile-first with bottom nav.
Lovable Cloud enabled. Auth via email/phone. Storage: music, covers, avatars buckets.
Admin system uses user_roles table with has_role() security definer function.
