# Project Memory

## Core
Sudagospel.net — gospel music platform for South Sudan. Dark purple/black bg with gold accents.
Primary gold (#D4A017 / 45 90% 55%), secondary purple (270 60% 50%), deep dark bg (260 20% 5%).
Figtree headings, DM Sans body. Mobile-first with bottom nav. Premium Spotify/Audiomack style.
Lovable Cloud enabled. Auth via email/phone. Storage: music, covers, avatars buckets.
Admin system uses user_roles table with has_role() security definer function.
